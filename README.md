# How to start
django react 기술 기반 프로젝트
1. git clone THIS PROJECT
## CLIENT - REACT
1. cd PROEJCT/client
2. npm install
3. Open Terminal
4. npm run watch

## SERVER - DJANGO
1. cd PROJECT
2. pip install -r requirements.txt
3. Open another Termianl
4. python manage.py runserver
